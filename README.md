## TGulp，实现前端工作自动化流程

### 基本条件
+ node.js
+ gulp
+ browser-sync:浏览器自动刷新
+ gulp-sass： css预处理器
+ gulp-concat： 合并
+ gulp-rename： 重命名
+ gulp-jshint: js查错
+ gulp-jshint-stylish : js差错提示
+ gulp-gulify： js压缩
+ gulp-minify-css：css压缩
+ gulp-imagemin： 图片压缩
+ gulp-spritemith :雪碧图
+ bower： 依赖管理
+ gulpfile.js

### 流程

src(js, sass, images)==>--(js, css,images.min)==>(all.min.js, all.min.css, images)dist

待细化和商榷
